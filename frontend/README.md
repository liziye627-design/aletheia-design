# Frontend Runtime Guide

## Current Runtime Mainline
- New React runtime (default): `frontend/src/main.tsx` + `frontend/src/App.tsx`
- Current runtime entry: `frontend/src/main.tsx`

## Local Run
```bash
cd frontend
npm install
npm run dev
```

Default frontend URL: `http://localhost:5173`

## Build
```bash
cd frontend
npm run build
```

## Unit Tests
```bash
cd frontend
npm run test:unit
```

## E2E (optional)
```bash
cd frontend
npm run test:e2e
```
